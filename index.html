Datensicherung & Backup
✓ Automatisches Backup aktiv
Ihre Daten werden in Supabase (Zürich, Schweiz) gespeichert und täglich automatisch gesichert.
Backup-Frequenz: Täglich, 2:00 Uhr
Aufbewahrung: 30 Tage
Standort: Supabase EU-Central-2, Zürich CH
Verschlüsselung: AES-256 at rest, TLS 1.3 in transit
Für manuelles Backup: Supabase Dashboard → Project → Backups
Supabase Dashboard öffnen ↗
E-Mail / Benutzer-Einladungen
⚠ E-Mail-Versand konfigurieren
Damit Benutzer Einladungs-E-Mails erhalten, muss SMTP in Supabase konfiguriert werden.
So aktivieren Sie E-Mail-Versand:
  1. Supabase Dashboard öffnen
  2. Authentication → Settings → SMTP Settings
  3. SMTP aktivieren und Infomaniak/Gmail/SendGrid eintragen
  4. Absender-E-Mail: noreply@ajconnect.ch
Empfehlung für Schweiz: Infomaniak Newsletter oder Brevo (ehemals Sendinblue)
